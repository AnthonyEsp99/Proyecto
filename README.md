# Simulación Braquistócrona 3D en Python

## Descripción
Simulación 3D interactiva del problema de la braquistócrona (curva de descenso más rápido) con física realista, texturas y reflejos en tiempo real.

## Características Principales
-  Tres tipos de curvas: Recta, Parábola, Cicloide
-  Física realista con gravedad y fricción
-  Texturas configurables (Hierro/Madera/Plástico)
-  Reflejos en tiempo real usando Cubemaps
-  Sistema de cámaras 3D interactivo
-  Panel de información en tiempo real
-  Configuración dinámica de alturas

## Requisitos
- Python 3.8+
- PyGame
- PyOpenGL
- NumPy
